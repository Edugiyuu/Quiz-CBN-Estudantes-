## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:

- Instalar a versão mais recente de `<vs code>`
- Ter uma máquina `<Windows / Linux / Mac>`.
- Instalar o programa `<Git>`
- Ter a extensão`<Life server>` "opcional".

## 🏃‍♂️‍➡️ Passo a passo

- 1° Passo: Entrar no `<vs code>`.
- 2° Passo: Abrir o repositorio <a href="`https://github.com/NicolasLeao12/Quiz-CBN.git">Quiz CBN</a>
- 3° Passo: Copiar o link do repositorio.
- 4° Passo: Abrir um terminal e digitar "git clone https://github.com/NicolasLeao12/Quiz-CBN.git"
